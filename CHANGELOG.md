# Changelog

All notable changes to the Alabastro theme will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2026-01-08

### Added
- Icon for marketplace listing
- Screenshots showcasing theme with multiple programming languages:
  - Python, React/JSX, TypeScript, Rust, HTML, CSS
  - Full editor UI overview
- Gallery section in README.md displaying all screenshots

### Changed
- Enhanced README.md with visual examples
- Improved marketplace presentation

## [1.0.0] - 2026-01-07

### Added
- Initial release of Alabastro color theme
- Dark variation of the minimal Alabaster syntax highlighting scheme with enhanced colors
- Comprehensive syntax highlighting for 40+ programming languages including:
  - Haskell, Python, Rust, Go, JavaScript/TypeScript
  - Elm, Clojure, Java, C/C++, C#, Ruby, PHP
  - CSS, HTML, JSON, YAML, Markdown, and many more
- Complete UI customization (editor, terminal, sidebar, status bar)
- Semantic token color support for modern VS Code features
- Terminal ANSI color configuration
- Dark theme optimized for eye comfort and extended coding sessions

### Features
- Minimal yet semantically rich color palette
- Consistent color usage across all elements
- Based on Alabaster's philosophy of focused, non-distracting syntax highlighting
- Fully customizable through VS Code settings
